# от поддержки DIVAO для MAX Messenger

## становка
1. `npm install`
2. Создай .env файл с токеном: `BOT_TOKEN=твой_токен`
3. `node bot.js`

## олучение токена
1. арегистрируй организацию на platform.max.ru
2. Создай бота, пройди модерацию
3. озьми токен в разделе "нтеграция"

## оманды бота
- /start - приветствие
- /activation - проблемы с Windows
- /performance - ноутбук греется/тормозит
- /contact - контакты
- /help - справка
